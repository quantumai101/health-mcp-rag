ProductRequirementsAgent

OpsSREAgent

2.7 Security-by-design: prompt protection, content filtering, access controls
Pattern: Azure GenAIOps security guidance + agent orchestration patterns.learn.microsoft+1

With SecurityGuardAgent:

Prompt protection:

Generate prompt templates that:

Enforce instructions hierarchy.

Use system prompts that resist prompt injection.

Restrict tool use to allowed actions.

Validate prompts before they go to Azure OpenAI (e.g., block exfiltration patterns).[learn.microsoft]

Content filtering:

Use Azure built‑in safety filters and/or custom classifiers.

Wrap all responses through a moderation stage agent that can redact or rephrase before returning to clients.[learn.microsoft]

Access control:

Enforce authz via:

Azure AD / Entra ID tokens.

Tenant-scoped data access in RAG queries.

Unity Catalog permission sets for models, features, and tables, allowing row-level and column-level security.learn.microsoft+1

Store credentials and keys in Key Vault; inject via managed identities, not env plaintext.[learn.microsoft]

Auditability:

Log who asked what, what sources were retrieved, which model answered, and which version, into MLflow / Delta for review.docs.azure+2

This covers: “Apply security-by-design practices including prompt protection, content filtering, access controls” and supports regulated/data‑sensitive environments.
